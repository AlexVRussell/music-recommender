## Solution

Felt like the most elegant solution was to make a js file for each of the students I selected to have personal pages.
The main gradebook is built by looping over arrays, calculating averages with helper functions, and inserting rows into the table.
The detail pages use arrays to show assessment grades and weighted averages, with missing grades handled as N/A.
