## Solution

When someone submits the account form, check the email, username, and password.
If one of the fields is filled out incorrectly, we display a clear message and marks that input in red.
If correctly submitted, it saves the new username and password and shows a success message.

When someone logs in, we check if the username exists and if the password matches in the map.
If either one is wrong, it shows what failed. Else we have a successful login.
